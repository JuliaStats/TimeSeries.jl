Array indexing

Array indexing

Indexing out a time series is done with common bracketing semantics.

Integers

ExampleDescriptionIndexing value
[1]First row of data onlysingle integer
[1:3]First through third row onlyinteger range
[1:2:10]Odd row between first to tenth rowinteger range with step
[[1:3; 8]]First through third row and eight rowinteger range & single integer

Examples in REPL:

julia> using MarketData

julia> ohlc[1]
1x4 TimeSeries.TimeArray{Float64,2,Date,Array{Float64,2}} 2000-01-03 to 2000-01-03

Date and DateTime

ExampleDescriptionIndexing value
[Date(2000, 1, 3)]The row containing Jan 3, 2000 timestampsingle Date
[[Date(2000, 1, 3), Date(2000, 2, 4)]]The rows containing Jan 3 & Jan 4, 2000multiple Dates
[Date(2000, 1, 3):Date(2000, 2, 4)]The rows between Jan 1, 2000 & Feb 1, 2000range of Date

Mixed approach

ExampleDescriptionIndexing value
["Open"][1:3]"Open" column & first 3 rowssingle string & integer range
["Open"][Date(2000, 1, 3)]"Open" column and Jan 3, 2000single string & Date

julia> using MarketData

julia> ohlc["Open"][1:3]

│            │ Open   │
├────────────┼────────┤
│ 2000-01-03 │ 104.88 │
│ 2000-01-04 │ 108.25 │
│ 2000-01-05 │ 103.75 │
